Data was generated by the Old SignalSnap using:

```python
with h5py.File("./tests/test_data/datasets/5Qubit_short_data.h5", "r") as f:
    X_test = f["/X_test"][:]

signal_channel_0 = X_test[:1000, :, 0].reshape(-1)
signal_channel_1 = X_test[:1000, :, 1].reshape(-1)

config1 = DataImportConfig(data=signal_channel_0)
config2 = DataImportConfig(data=signal_channel_1)
selected_data = [0,1] # because we want to evaluate both arrays. 
                      # If only one was of interest, we could have used selected_data = [0]

sconfig = SpectrumConfig(
    dt=2., 
    f_min=0., 
    f_max=0.25, 
    s3_calc='1/4',
    f_unit='GHz', 
    backend='cuda', 
    order_in=[1, 2, 3, 4], 
    spectrum_size=100, 
    show_first_frame=False
)

cconfig = CrossConfig(auto_corr=True)

scalc = SpectrumCalculator(sconfig, cconfig, [config1, config2], selected=selected_data)
scalc.calc_spec()

np.savez_compressed(
    "5Qubit_short_data_auto_corr.npz", 
    spectra=scalc.s, 
    error=scalc.s_err, 
    freqs=scalc.freq
)
```


``5Qubit_short_data.h5`` stores measurements performed on a 5 Qubit Chip (see DOI: 10.1103/PhysRevApplied.17.014024).
The dataset /X_test stores the results and has shape (2000, 1000, 2).
The in-phase and quadrature components are both stored in the dataset /X_test and can be disitinguished by the last index.
The dataset is comprised of 2000 measurements with 1000 sample each. The time between two samples is ``dt=2ns``. 
Between each 1000 sample the chip is initialized again.
